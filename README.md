# 🎨 Guía de Diseño de Interfaces Web

Una guía completa, práctica y moderna para estudiantes de Ingeniería Informática. Aprende a crear interfaces profesionales usando HTML, CSS y JavaScript.

## 🌐 Ver en línea

Esta guía está publicada en GitHub Pages:

**https://TU-USUARIO.github.io/design-web-guide/**

## 📚 Contenido

La guía está dividida en 5 partes:

### Parte 1: Fundamentos UX/UI
- Introducción al diseño de interfaces
- UX vs UI: diferencias clave
- Principios fundamentales de diseño
- Las 10 Heurísticas de Nielsen
- Psicología del color

### Parte 2: HTML, CSS y Responsive
- HTML semántico
- Formularios accesibles
- CSS moderno (variables, Flexbox, Grid)
- Diseño responsive y mobile-first

### Parte 3: Componentes y Librerías
- Controles comunes (botones, tablas, modales)
- Controles mejorados con JavaScript
- Librerías recomendadas (Bootstrap, Flatpickr, DataTables, SweetAlert2)
- Patrones de diseño comunes (CRUD, Dashboard, Wizard)

### Parte 4: Temas Avanzados
- Accesibilidad Web (WCAG)
- Mobile-first profundo
- Modo oscuro / claro
- Optimización de rendimiento

### Parte 5: Extras y Recursos
- Antes y después (casos reales)
- Errores comunes en interfaces administrativas
- Glosario UX/UI completo
- Recursos recomendados
- Guía paso a paso para GitHub Pages

## 💻 Tecnologías utilizadas

- HTML5 semántico
- CSS3 (variables, Grid, Flexbox)
- JavaScript Vanilla
- Prism.js (syntax highlighting)
- Font Awesome (iconos)
- Google Fonts (Inter, Fira Code)
