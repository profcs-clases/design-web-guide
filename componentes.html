Controles Mejorados con JavaScript

Lleva tu interfaz al siguiente nivel

Los controles nativos del navegador son limitados. Con JavaScript puedes crear experiencias mucho más ricas y amigables.

Validación en tiempo real

JavaScript 
const input = document.getElementById('cedula');
const errorMsg = document.querySelector('.error-message');

input.addEventListener('input', (e) => {
    const value = e.target.value;
    const pattern = /^(V|E)-\d{7,8}$/;
    
    if (value === '') {
        input.classList.remove('is-valid', 'is-invalid');
        errorMsg.textContent = '';
    } else if (pattern.test(value)) {
        input.classList.remove('is-invalid');
        input.classList.add('is-valid');
        errorMsg.textContent = '';
    } else {
        input.classList.remove('is-valid');
        input.classList.add('is-invalid');
        errorMsg.textContent = 'Formato: V-12345678 o E-12345678';
    }
});

Máscara de entrada para cédula venezolana

JavaScript 
function maskCedula(input) {
    input.addEventListener('input', (e) => {
        let value = e.target.value.replace(/\D/g, ''); // Solo números
        
        if (value.length === 0) {
            e.target.value = '';
            return;
        }
        
        // Detectar si empieza con V o E
        const firstChar = value.charAt(0);
        let prefix = 'V';
        
        if (firstChar === '1' || firstChar === '2') {
            prefix = 'V';
        } else if (firstChar === '8' || firstChar === '9') {
            prefix = 'E';
        }
        
        // Limitar a 8 dígitos después del prefijo
        value = value.slice(0, 8);
        e.target.value = `${prefix}-${value}`;
    });
}

maskCedula(document.getElementById('cedula'));

Máscara para teléfono venezolano

JavaScript 
function maskTelefono(input) {
    input.addEventListener('input', (e) => {
        let value = e.target.value.replace(/\D/g, '');
        
        // Limitar a 11 dígitos (0414-1234567)
        value = value.slice(0, 11);
        
        if (value.length <= 4) {
            e.target.value = value;
        } else {
            e.target.value = value.slice(0, 4) + '-' + value.slice(4);
        }
    });
}

maskTelefono(document.getElementById('telefono'));

Selects dependientes (Estado → Ciudad)

JavaScript 
const estados = {
    'distrito-capital': ['Libertador'],
    'miranda': ['Sucre', 'Plaza', 'Chacao', 'Baruta'],
    'zulia': ['Maracaibo', 'San Francisco', 'Cabimas'],
    'carabobo': ['Valencia', 'Puerto Cabello', 'Naguanagua']
};

const estadoSelect = document.getElementById('estado');
const ciudadSelect = document.getElementById('ciudad');

estadoSelect.addEventListener('change', (e) => {
    const estado = e.target.value;
    
    // Limpiar ciudades anteriores
    ciudadSelect.innerHTML = '';
    
    if (estado && estados[estado]) {
        estados[estado].forEach(ciudad => {
            const option = document.createElement('option');
            option.value = ciudad.toLowerCase();
            option.textContent = ciudad;
            ciudadSelect.appendChild(option);
        });
        ciudadSelect.disabled = false;
    } else {
        ciudadSelect.disabled = true;
    }
});

Modal funcional (clase reusable)

JavaScript 
class Modal {
    constructor(id) {
        this.overlay = document.getElementById(id);
        this.modal = this.overlay.querySelector('.modal');
        this.isOpen = false;
        
        this.init();
    }
    
    init() {
        // Cerrar al hacer clic fuera
        this.overlay.addEventListener('click', (e) => {
            if (e.target === this.overlay) this.close();
        });
        
        // Cerrar con tecla ESC
        document.addEventListener('keydown', (e) => {
            if (e.key === 'Escape' && this.isOpen) this.close();
        });
        
        // Botones de cerrar
        this.overlay.querySelectorAll('[data-close]').forEach(btn => {
            btn.addEventListener('click', () => this.close());
        });
    }
    
    open() {
        this.overlay.classList.add('active');
        this.isOpen = true;
        document.body.style.overflow = 'hidden';
    }
    
    close() {
        this.overlay.classList.remove('active');
        this.isOpen = false;
        document.body.style.overflow = '';
    }
}

// Uso
const modal = new Modal('mi-modal');
document.getElementById('abrir-modal').addEventListener('click', () => {
    modal.open();
});

Confirmación elegante antes de eliminar

JavaScript 
function confirmarEliminacion(id, nombre) {
    const confirmed = confirm(
        `¿Eliminar "${nombre}"?\n\nEsta acción no se puede deshacer.`
    );
    
    if (confirmed) {
        // Lógica de eliminación
        fetch(`/api/items/${id}`, { method: 'DELETE' })
            .then(res => res.json())
            .then(data => {
                showToast('Registro eliminado correctamente', 'success');
                document.querySelector(`[data-id="${id}"]`).remove();
            });
    }
}

function showToast(message, type = 'success') {
    const toast = document.createElement('div');
    toast.className = `toast toast-${type}`;
    toast.textContent = message;
    document.body.appendChild(toast);
    
    setTimeout(() => {
        toast.style.animation = 'slideInRight 0.3s reverse';
        setTimeout(() => toast.remove(), 300);
    }, 3000);
}
Regla de oro: Siempre confirma antes de acciones destructivas. El usuario puede hacer clic por error. Un "¿Estás seguro?" bien puesto puede evitar desastres.

03

Librerías Recomendadas

No reinventes la rueda

Existen librerías probadas que resuelven problemas comunes. Úsalas sabiamente: no cargues 10 librerías si solo necesitas una.

Frameworks CSS

Bootstrap

El más usado

Sistema de grid, componentes pre-diseñados, ideal para prototipar rápido. Muy popular en entornos empresariales venezolanos.

~170KB (con JS)

Tailwind CSS

Utility-first

Clases utilitarias, altamente personalizable, cero estilos por defecto. Curva de aprendizaje alta pero muy flexible.

~10KB (producción)

CSS Puro

Control total

Sin dependencias, rendimiento óptimo, aprendizaje profundo. Ideal cuando quieres control absoluto.

0KB (solo tu código)

Librerías JavaScript ligeras

Librería Propósito Tamaño
Select2 Selects con búsqueda y tags ~70KB
Flatpickr Datepicker ligero y personalizable ~40KB
DataTables Tablas con búsqueda, orden, paginación ~120KB
SweetAlert2 Alertas y confirmaciones hermosas ~50KB
Chart.js Gráficos simples y responsive ~200KB
Quill.js Editor de texto enriquecido ~500KB
Inputmask Máscaras de entrada avanzadas ~40KB
Toastify Notificaciones tipo toast ~10KB

Manejo de estados

Toda interfaz debe manejar 4 estados: vacío, cargando, error y éxito.

Vacío

Invita a crear el primer registro

Cargando

Skeleton loaders o spinners

Error

Mensaje claro con opción de reintentar

Éxito

Feedback visual y redirección suave
